# Security Policy

## Supported Versions

The following versions of BBC-AOS are currently supported with security updates:

| Version | Supported |
| :--- | :---: |
| 1.0.x | Yes |

## Reporting a Vulnerability

We take the security and integrity of BBC-AOS very seriously. If you discover a security vulnerability, please report it immediately:
1. Do **not** open a public issue.
2. Email your report to: `security@bbc-aos.local`.
3. Include detailed reproduction steps, exploit payloads, and environment configurations.

We will acknowledge your report within 48 hours and coordinate a public disclosure window after a patch has been released.
